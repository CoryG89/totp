# totp

Simple command line [TOTP](https://en.wikipedia.org/wiki/Time-based_One-time_Password_Algorithm) generator.

## Usage

```
$ totp github.com/defunkt
123789
```

## Installing

```
$ git clone https://github.com/josh/totp
$ cd totp/
$ make install
```
